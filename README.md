# PHP Router Template 

A PHP Template with pre built functions and utlities.


## Pre Built

- Router
- Middleware Auth
- Service Container
- Database
- Validator
- Response
- Sessions Handler
- Login & Logout & Register
- Controller
- Views
- Partials
- dd function for debuging



## Requirements

- TailwindCSS
- PHP 8+
- MySQL DB
